## 🆘 Troubleshooting

### Server not starting?
```bash
rm -rf .next
npm run dev
```

### TypeScript errors?
They're just warnings. App still works! The errors you're seeing are IDE caching issues.

### Form not working?
Check that `.env.local` is configured with Supabase credentials.
